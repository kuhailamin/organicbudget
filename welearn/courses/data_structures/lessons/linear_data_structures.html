Linear Data Structures
  • Linear data structures can be defined as data structures with items that are physically laid out next to each other.
  • Items in vectors are stored next to one another on the physical memory.
  • On the other hand, items in lists are virtually linked to one another by means of pointers.
  • The advantages of linear data structures are: they are relatively easy to understand, and they have many useful applications.
  • The disadvantages of linear data structures are: it may be slow to search , add, delete, or modify data.
